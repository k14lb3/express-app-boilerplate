# Express App Boilerplate

Express app boilerplate.

## Installation

```
git clone --depth=1 https://github.com/k14lb3/express-app-boilerplate.git <project-name>
cd <project-name>
rm -rf .git
git init .
```

## Usage

Install the dependencies:

```
yarn install
```

Run the development server:

```
yarn dev
```

Build for production:

```
yarn build
```
